# CS 271 - Homework 6 - Assembly 3

## Makers

### Build

```shell
make
```

### Rebuild

```shell
make rebuild
```

### Clean

```shell
make clean
```

## Debug

Build, run kdbg, select the source and the executable if you didn't start it by pointing at the executable.

```shell
kdbg hw6
```
